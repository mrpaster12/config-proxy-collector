# 🚀 Config & Proxy Collector

[![WebSite](https://img.shields.io/badge/Website-Visit_Now-blue?style=for-the-badge&logo=github)](https://mrpaster12.github.io/config-proxy-collector/)

به پروژه **جمع‌آوری‌کننده پروکسی و کانفیگ** خوش آمدید! 🌟
این پروژه یک ابزار (و وب‌سایت) خودکار است که با جستجو در منابع مختلف، جدیدترین و سالم‌ترین کانفیگ‌ها و پروکسی‌ها را برای دور زدن محدودیت‌های اینترنتی جمع‌آوری کرده و در اختیار شما قرار می‌دهد.

## ✨ ویژگی‌های پروژه

- **جمع‌آوری خودکار:** استخراج مداوم و خودکار کانفیگ‌ها از منابع معتبر.
- **پشتیبانی از پروتکل‌های مختلف:** پشتیبانی از انواع کانفیگ‌ها مانند `Vless`, `Vmess`, `Trojan`, `Shadowsocks` و...
- **رایگان و متن‌باز:** استفاده از این سرویس برای تمامی کاربران کاملاً رایگان است.
- **بروزرسانی مداوم:** لیست پروکسی‌ها در بازه‌های زمانی کوتاه به‌روز می‌شوند تا بیشترین پایداری را داشته باشند.

## 🔗 نحوه استفاده

برای استفاده از پروکسی‌ها و کانفیگ‌های جمع‌آوری شده، کافیست به وب‌سایت پروژه مراجعه کنید:

🌐 **[ورود به صفحه کانفیگ‌ها (کلیک کنید)](https://mrpaster12.github.io/config-proxy-collector/)**

شما می‌توانید لینک‌های اشتراک (Subscription Links) یا کانفیگ‌های تکی را از سایت کپی کرده و در برنامه‌های کلاینت خود وارد کنید.

## 📱 برنامه‌های پیشنهادی (Clients)

بسته به سیستم‌عامل خود می‌توانید از برنامه‌های زیر برای اجرای کانفیگ‌ها استفاده کنید:

* **اندروید (Android):** v2rayNG / NekoBox / MahsaNG
* **ویندوز (Windows):** v2rayN / NekoRay / Hiddify-Next
* **آی‌اواس (iOS):** V2Box / Streisand / Shadowrocket / Foxray
* **مک و لینوکس (Mac & Linux):** v2rayU / NekoRay / Hiddify-Next

## ⚠️ سلب مسئولیت
این پروژه صرفاً با هدف کمک به دسترسی آزاد به اطلاعات و دور زدن سانسور اینترنت ایجاد شده است. کانفیگ‌ها از منابع عمومی در اینترنت جمع‌آوری می‌شوند و ما هیچ کنترل مستقیم یا تضمینی روی آپتایم بودن همیشگی یا امنیت سرورهای مبدأ نداریم. لطفاً در هنگام استفاده از شبکه‌های عمومی، نکات امنیتی (مانند استفاده از HTTPS) را رعایت کنید.
